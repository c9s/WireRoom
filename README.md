WireRoom
===========


Requirement
-----------

- node.js
- redis
- mongodb
- coffee-script

Features
--------

* Jenkins Notification Integration
* GitHub PostReceive Integration
* Git Commit Notification (for private git repository)


Supported Browsers
------------------
* iOS6 (iPad or iPhone)
* Safari
* Google Chrome
* FireFox

Installation
------------

    npm install


Run
---

    coffee app.coffee
